## 6098 (https://codeup.kr/problem.php?id=6098)
풀이


```python
x, y = 1, 1
# 10*10 배열 생성
maze=[[]*10]*10

# 배열에 값 입력
for i in range(10):
    maze[i]=list(map(int,input().split()))

while maze[x][y] == 0:
    maze[x][y]=9
    if maze[x][y+1] == 0 or maze[x][y+1] == 2:
        y += 1
    else:
        x += 1

if maze[x][y] == 2:
    maze[x][y] = 9
    
# 배열 값 출력
for i in maze:
    print(' '.join(map(str, i)))
```

<br>

- 메모리 : 33764
- 시간 : 20
