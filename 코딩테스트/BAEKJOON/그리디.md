## 전자레인지(Bronze 4) (https://www.acmicpc.net/problem/10162)
풀이

```python
t = int(input())
A=B=C=0

if(t % 10 != 0):
    print("-1")
else:
    A = t // 300
    t %= 300
    B = t // 60
    t %= 60
    C = t // 10
    t %= 10
    print("{} {} {}".format(A,B,C))
```

- 시간 : 68

## 세탁소 사장 동혁 (Bronze 3) (https://www.acmicpc.net/problem/2720)
풀이

```python
for _ in range(int(input())):
    C = int(input())
    
    Q = C // 25
    C %= 25
    D = C // 10
    C %= 10
    N = C // 5
    C %= 5
    P = C // 1
    C %= 1
    
    print("{} {} {} {}".format(Q,D,N,P))
```
